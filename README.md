# School web application

- generate and fill secrets listed in `school-web-app.env`:

```
make local-env
```

- to build and run the `docker-compose` in one step simply run:

```
make
```

- to build:

```
make build
```

- to run:

```
make run
```

- to visit the webpage go to `localhost:FRONTEND_PORT`
